# 객체 탐지 프로젝트

## 프로젝트 사전 설정

### 콘다 가상환경 생성

```bash
conda env create -f environment.yaml
```

### 콘다 가상환경 활성화

```bash
conda activate cnn
```

## 프로젝트 실행

### 디버깅

```bash
gradio main.py
```

### 실제 어플리케이션

```bash
python main.py
```
